# 🧊 Mini 3D Engine in JavaScript

A small educational project focused on building a miniature 3D rendering engine from scratch. 
Currently, the project allows for rendering and interactively rotating a 3D cube in real-time. 
The main goal was to understand the mathematics behind 3D graphics and perspective projection without using heavy external libraries.

## 🛠 Tech Stack
* **Language:** Vanilla JavaScript
* **Rendering:** HTML5 Canvas
* **Frontend:** HTML, CSS

## ✨ Key Features
* Rendering 3D vertices and edges on a 2D canvas.
* Calculation of perspective projection (3D to 2D).
* Smooth rotation functionality across multiple axes.

## 💻 How to Run Locally
This project uses ES6 Modules, so it requires a local web server to run properly.
1. `git clone [your_repo_link]`
2. Serve the project folder using your preferred local server (e.g., the **Live Server** extension in VS Code, `npx http-server`, or `python3 -m http.server`).
